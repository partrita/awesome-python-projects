## 기능
- [x] 사용자 인증
- [x] 검색
- [x] 관리자로부터 실시간 업데이트
- [x] 삭제
- [x] 성별에 따른 필터링
- [x] 프로필 이미지



## 실행 방법
```
git clone https://github.com/saadhaxxan/Contact-Manager-Django.git
cd Contact-Manager-Django
python manage.py runserver
```
## 관리자 패널에 접근하기
```
python manage.py createsuperuser
```
